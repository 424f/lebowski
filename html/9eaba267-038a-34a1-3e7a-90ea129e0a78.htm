The ITextContext type exposes the following members.

Methods

  NameDescription
Delete
Performs a deletion operation on the context.
Insert
Performs a insertion operation on the context.
Invoke
Invokes the given action in a thread that has read-write access to this context. If you are not operating on e.g. a UI control, you can just implement this method by calling the delegate directly.
Refresh
Refreshes the context, making sure that changes are correctly reflected in any UI associated with it.
SetRemoteSelection
Sets the remote selection of the user identified by

Field Value

siteIdentifier. Some text context implementations might support displaying those remote selections to achieve better interaction awareness.
SetSelection
Sets the local selection.

SelectionStart
