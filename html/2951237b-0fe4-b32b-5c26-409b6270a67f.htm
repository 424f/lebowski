Classes

AbstractTextContext
Provides functionality that most implementing classes of ITextContext share.
ChangeEventArgs
Provides data for the Change event.
DeleteEventArgs
Provides data for the Delete event. This is used for algorithms that are based on atomic operations (e.g. operational transformations).
InsertEventArgs
Provides data for the Insert event. This is used for algorithms that are based on atomic operations (e.g. operational transformations).
StringTextContext
A text context that does not have any GUI representation, but just applies operations to a string instead. This is useful for performing text operations on a shared context.
TextBoxTextContext
TextBoxTextContext is a text context that is linked with a standard WinForms TextBox. It does not currently support display of remote selections.
TextEditorTextContext
TextBoxTextContext is a text context that is linked with a standard ICSharpCode.TextEditor that supports syntax hightlighting, display of remote selections currently support display of remote selections.
