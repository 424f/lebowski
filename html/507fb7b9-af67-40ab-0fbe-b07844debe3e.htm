TokenState Enumeration
See Also  Send Feedback
Indicates whether a site currently holds the token. In the differential synchronization between two sites, only one site ever holds a token, which allows it to distribute its changes back to the other site. If it notices local changes but does not hold the token, it has to first request it from the other site.
